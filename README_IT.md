# Agharti Esplorazioni

Un sito web interattivo per una guida escursionistica professionale che opera nelle regioni dell’Aspromonte e delle Serre in Calabria, Italia.

Il sito funge sia da vetrina pubblica per le attività di escursionismo della guida, sia da blog reso funzionale tramite Supabase.

---

## Funzionalità

- Design interattivo per dispositivi desktop e mobile
- Presentazione della guida e delle attività escursionistiche
- Sezione escursioni con galleria immagini
- Sezione blog per articoli
- Autenticazione Supabase per amministratori
- Pannello admin per la creazione e l’eliminazione dei post
- Supporto per il caricamento multiplo di immagini
- Carousel di immagini con navigazione touch swipe
- Visualizzazione delle immagini a schermo intero
- Integrazione contatti e social media
